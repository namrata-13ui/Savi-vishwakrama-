# Python Syntax Notes

## Python Indentation
- Python uses indentation instead of curly braces {}.
- The standard indentation is 4 spaces.
- Proper indentation makes the code readable.
- Incorrect indentation results in an IndentationError.

## Variables
Variables are used to store data.
Python automatically determines the data type of a variable.

Example:
```python
name = "Namrata"
age = 20
```

## Data Types
Python has several built-in data types:

- String (str): Stores text.
- Integer (int): Stores whole numbers.
- Float (float): Stores decimal numbers.
- Boolean (bool): Stores True or False values.

## Input and Output

Input:
```python
name = input("Enter your name: ")
```

Output:
```python
print(name)
```
