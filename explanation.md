# Error Handling & Debugging Basics

## Objective

The purpose of this task is to understand how Python handles runtime errors using exception handling.

## Exceptions Used

### 1. ZeroDivisionError
Occurs when a number is divided by zero.

### 2. ValueError
Occurs when the input type is incorrect.

### 3. FileNotFoundError
Occurs when the specified file is not found.

### 4. IndexError
Occurs when accessing an invalid list index.

## Keywords

- try
- except
- finally

## Conclusion

Exception handling prevents programs from crashing and provides meaningful error messages to users.
